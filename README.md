# SECOM OWEL カルテコピー

SECOM OWEL電子カルテの情報を範囲選択してコピー・ダウンロードするChrome拡張機能

## 機能

- 全ページ自動読み込み（1〜最終ページまで）
- - 範囲選択でエントリーを指定
  - - テキスト / JSON 形式で出力
    - - クリップボードにコピー または ファイルダウンロード
      - - 患者情報・バイタル情報の含有を選択可能
       
        - ## インストール方法
       
        - 1. このリポジトリをダウンロード（Code → Download ZIP）
          2. 2. ZIPを解凍
             3. 3. Chromeで `chrome://extensions/` を開く
                4. 4. 右上の「デベロッパーモード」をON
                   5. 5. 「パッケージ化されていない拡張機能を読み込む」をクリック
                      6. 6. 解凍したフォルダを選択
                        
                         7. ## 使い方
                        
                         8. 1. SECOM OWELにログインし、患者のカルテページを開く
                            2. 2. 拡張機能アイコンをクリック
                               3. 3. 「🔄 全ページを読み込む」をクリック（約30秒〜1分）
                                  4. 4. 範囲を選択（開始・終了のエントリーを選択）
                                     5. 5. 出力形式（テキスト/JSON）を選択
                                        6. 6. 「📋 クリップボードにコピー」または「💾 ファイルをダウンロード」をクリック
                                          
                                           7. ## ファイル構成
                                          
                                           8. - `manifest.json` - 拡張機能の設定
                                              - - `popup.html` - ポップアップUI
                                                - - `popup.js` - メインロジック
                                                  - - `styles.css` - スタイル
                                                   
                                                    - ## 注意事項
                                                   
                                                    - - 全ページ読み込み中はブラウザのタブを切り替えないでください
                                                      - - 読み込み時間は約1.5秒 × ページ数
